# Personality Test Lab legal site

Upload these files to a public static host.

Recommended URL after GitHub Pages setup:

https://<your-github-username>.github.io/personality-test-lab-legal/privacy.html

Use that URL in:
- `.env` -> `EXPO_PUBLIC_PRIVACY_POLICY_URL`
- Google Play Console -> App content / Privacy policy
- AdMob privacy message settings

## GitHub Pages quick setup

1. Create a public GitHub repository named `personality-test-lab-legal`.
2. Create a folder named `docs`.
3. Upload `privacy.html`, `terms.html`, and `index.html` into `docs`.
4. Go to repository Settings -> Pages.
5. Source: Deploy from a branch.
6. Branch: `main`, folder: `/docs`.
7. Save and wait for Pages to publish.
